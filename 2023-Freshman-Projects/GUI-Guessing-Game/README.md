# GUI Number Guessing Game (2023)

**Course:** Computational Thinking (Freshman Year)  
**Tech Stack:** Python (Tkinter/PyQt)

### Description
A graphical interface application where users guess a randomly generated number within a constrained range. This project was my first implementation of:
* **Event-Driven Programming:** Handling button clicks and user input.
* **Input Validation:** Ensuring user entries are integers within bounds.
* **GUI Layout Management:** Organizing widgets effectively.

### How to Run
```bash
python main.py
